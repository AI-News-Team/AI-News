# Tests for Mocha and Cypress

## For running the Cypress tests

>Navgate to the cypress folder
>
> ```shell
>cd test/cypress
> ```
>
>then to run either:
>
> ```shell
>npx cypress open
> ```
>
>or:
>
> ```shell
>npx cypress run
> ```
>
## For Running Mocha Tests

>Navagate to the Mocha folder
>
> ```shell
>cd test/mocha
> ```
>
>then to run:
>
> ```shell
>npx mocha
> ```
>
